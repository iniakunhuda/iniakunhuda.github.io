Surabaya Zoo Information System



Year 2018
Built with Laravel, jQuery

Surabaya Zoo Information System is a web link between the Surabaya zoo and visitors. On this website there are animal galleries, event schedules, blogs, and visitors can buy tickets online here. In addition, there is an interactive map that will help visitors to determine the route that can be passed when exploring the zoo. The purpose of making this information system is to attract people to visit the Surabaya zoo by providing access to online ticket purchases, interactive zoo maps, and seeing what events are on that day.
